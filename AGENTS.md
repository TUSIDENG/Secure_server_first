Linux 系统管理员 Skills（适配 Cursor/Trae Agent）

核心定位：Agent 可直接调用的 Linux 运维技能集，聚焦非交互式脚本编写、详细操作指南，适配 Ubuntu/CentOS 跨平台，贴合堡垒机架构、系统安全加固、容器运维场景，输出可直接执行的命令/脚本及配套操作步骤，无需多余解释；安全相关技能可参考 GitHub 仓库：https://github.com/imthenachoman/How-To-Secure-A-Linux-Server。

一、系统基础操作（非交互式脚本+操作指南）

- 跨平台系统初始化：编写非交互式脚本，实现 Ubuntu（apt）/CentOS（yum/dnf）系统更新、依赖自动安装，脚本内集成系统版本自动识别逻辑，配套详细操作指南（执行步骤、异常处理）

- 用户与权限管理：编写非交互式用户管理脚本（创建普通用户、配置 sudo 权限/免密 sudo、用户删除/密码重置），配置文件权限（chmod/chown/ACL），输出完整操作指南，无需交互式输入

- 服务管理：编写非交互式脚本实现 systemd 服务启停、自启配置、状态排查（systemctl 系列命令），配套操作指南（脚本执行步骤、服务状态校验方法）

- 系统监控：编写监控脚本（进程、磁盘、内存、网络、日志排查），输出非交互式执行命令，配套操作指南（结果解读、异常排查步骤）

- 定时任务：编写非交互式 crontab 配置脚本，实现任务调度、日志输出设置，配套操作指南（脚本部署、任务校验、日志查看方法）

二、安全加固（核心技能，贴合堡垒机场景，参考指定GitHub仓库）

- SSH 加固：参考 https://github.com/imthenachoman/How-To-Secure-A-Linux-Server 内容，编写非交互式 SSH 加固脚本（密钥登录配置、禁止 root 直接登录、关闭密码登录、SSH 端口修改），输出完整操作指南，脚本可直接执行无需交互

- 防暴力破解：参考指定 GitHub 仓库安全规范，编写非交互式 fail2ban 安装、配置、启停脚本，适配 Ubuntu/CentOS 不同版本，配套操作指南（配置解读、状态校验）

- 防火墙配置：编写非交互式防火墙规则脚本（ufw 适配 Ubuntu、firewalld 适配 CentOS），实现端口开放/限制（22/80/443 等），一键生效，配套操作指南（规则查看、修改方法）

- 系统基线加固：参考指定 GitHub 仓库 CIS 基准要求，编写非交互式系统加固脚本（内核参数优化、危险服务关闭、PAM 密码策略配置），配套操作指南（加固后校验、回滚方法）

- 安全审计：参考指定 GitHub 仓库安全审计规范，编写非交互式脚本开启 sudo 操作日志、配置 systemd 审计（auditd），确保操作可追溯（适配堡垒机审计需求），配套操作指南（日志查看、审计分析步骤）

三、容器与云服务器运维（非交互式脚本+操作指南）

- Docker 运维：编写非交互式 Docker 安装脚本（配置国内镜像源）、镜像管理、容器启停/部署、docker-compose 编排脚本，配置镜像加速器，配套详细操作指南（脚本执行、容器状态校验）

- 云服务器适配：编写非交互式阿里云/腾讯云 ECS 初始化脚本（安全组配置、密钥下发），贴合堡垒机密钥托管场景，配套操作指南（脚本部署、密钥校验方法）

- 容器数据安全：编写非交互式容器数据持久化、镜像备份/恢复脚本，Vaultwarden 容器部署与配置（多用户、组织共享）脚本，配套操作指南（部署步骤、数据备份校验）

四、脚本编写（核心任务：非交互式脚本+操作指南）

- Bash Shell 脚本：核心任务，编写非交互式脚本（用户创建、系统加固、批量操作等），支持可配置参数（脚本内预设参数，无需交互式输入），配套完整操作指南（执行步骤、参数修改方法、异常处理）

- Python 基础脚本：编写非交互式 Python 脚本（日志分析、系统监控告警、自动备份、漏洞扫描等），输出可直接执行命令，配套操作指南（环境依赖、脚本部署、结果解读）

- 脚本优化：对编写的非交互式脚本进行去冗余、增加容错处理，确保可重复执行、适配 Ubuntu/CentOS 跨平台，配套脚本优化指南（优化要点、测试方法）

五、中间件与服务部署（非交互式脚本+操作指南）

- Web 服务：编写非交互式 Nginx/Apache 安装、配置、反向代理、HTTPS（SSL/Certbot）配置脚本，配套详细操作指南（部署步骤、服务校验、配置修改）

- 数据库运维：编写非交互式 MySQL/PostgreSQL 基础安装、备份、权限配置、慢查询优化脚本，配套操作指南（脚本执行、数据库状态校验、备份恢复步骤）

- 自建服务：编写非交互式 Vaultwarden（多用户密码管理）、监控面板部署脚本，生成一键部署命令，配套操作指南（部署步骤、服务配置、异常排查）

六、Agent 适配补充（关键）

- 核心输出：非交互式可执行脚本 + 详细操作指南，无需交互式输入，Agent 可直接调用脚本执行，或按操作指南输出对应命令

- 安全相关：所有安全加固脚本、操作指南，均参考 GitHub 仓库 https://github.com/imthenachoman/How-To-Secure-A-Linux-Server 内容，确保安全规范达标

- 跨平台适配：脚本内自动识别系统版本（Ubuntu/CentOS），输出对应适配命令/脚本，避免跨平台报错，操作指南同步区分不同系统操作差异

- 贴合堡垒机架构：所有脚本、操作均基于「普通用户+sudo」模式，禁止直接 root 操作，符合企业安全规范，操作指南明确标注权限要求
